# Unreal Engine binary uasset merge and diff tools integration into git

This is a set of scripts that add `git difftoll` and `git mergetool` for Unreal Engine .uasset files.
It can be useful for resolving conflicts when rebasing local branches.

The `/install.ps1 <Path\Tp\Git\bin\bash.exe>` script installs the merge and diff tool to be used in Windows git-bash prompt.
